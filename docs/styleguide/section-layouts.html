6布局 Layouts

布局

6.1Box（废弃）

使用表格布局（display: table），实现界面高度的自适应。.box元素（即父元素）要有高度，否则无法自动计算 注意： 这种方法在高版本的浏览器中有兼容性问题

由于 table-row 的百分比高度规范未定义，在IE10及以下的IE浏览器实现中，其内部元素无法应用父元素百分比高度 需要引入 tiny.js 修复此问题

Example 
<div class="box" style="height:300px;">
    <div style="height:30px;background:#ccc;"></div>
    <div class="box-row">
        <div class="box-cell">
            <div style="height:800px;background:#aaa;"></div>
        </div>
    </div>
    <div style="height:30px;background:#ccc;"></div>
</div>

6.2Flexbox

使用 flexbox 实现高度自适应的布局

兼容性

Example 
<div class="flexbox" style="height:300px;">
    <div style="height:30px;background:#ccc;"></div>
    <div class="grow">
        <div style="height:800px;background:#aaa;"></div>
    </div>
    <div style="height:30px;background:#ccc;"></div>
</div>

6.3Static Row

固定的栅格布局（常用在 flexbox 内部形成多栏自适应布局）

Example 
<div class="flexbox scroll-x" style="height:300px;">
    <div style="height:30px;background:#ccc;"></div>
    <div class="grow row-container">
        <div class="row row-static">
            <div class="col-static col-xs-3 no-padding">
                <div style="height:400px;background:#999;"></div>
            </div>
            <div class="col-static col-xs-9 col-xs-offset-3 no-padding">
                <div style="height:800px;background:#aaa;"></div>
            </div>
        </div>
    </div>
    <div style="height:30px;background:#ccc;"></div>
</div>
